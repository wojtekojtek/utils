# utils
### aka my scripts I use to quickly setup OSes to my own needs

## Usage
`./install.sh`<br>
That's all! Easy as a pie.<br>

## Supported Systems

- Raspberry Pi (Debian 11)<br>
- Ubuntu x64<br>
- Fedora x64<br>

## What it does
Automatically detects your OS and installs the tools **I** need. No configuration required.<br>
